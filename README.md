# Project Name & Pitch

Search for books

This web page allows users to search for books by Title, Author, or Genre. Th–µ app is built with React, TypeScript, and Css.

## Installation and Setup Instructions

Clone down this repository. You will need `node` and `npm` installed globally on your machine.

Installation:

`npm install`

To Start the Aplication:
`npm start`

To Visit App:
`localhost:3000`
